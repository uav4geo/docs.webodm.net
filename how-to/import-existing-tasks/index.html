Import Existing Tasks

Datasets processed outside the cloud interface, for example with WebODM Lightning's desktop client or any other WebODM setup, can be imported by following the steps below.

After import, links to a task can be shared with others.

Import from Desktop Client or WebODM​

info
  • All Assets will be smaller in size, but the archive will not include the original images used for processing as well as existing measurements.
  • Backup includes everything (it's an exact copy), but will take longer to upload and will consume more cloud space.
  1. Navigate to the cloud interface

Import from Lightning's Dashboard​

Results from the Lightning Dashboard can be imported if they have been processed within the last 14 days. After 14 days, the links there are no longer valid.
