Add Panoramas and Videos

After a task has been processed from the cloud interface, you can add additional Media files such as videos, 360° panoramas and images. These are then displayed on the map to provide additional visual context and information.

Uploading Media​

You can add a brief description or annotation to each file by switching to List View mode and pressing the + button.

Georeferencing Video Files

The program automatically attempts to extract georeferencing data from your videos. If your drone stores this information in a separate .SRT file, upload it alongside the video (e.g., upload both myvideo.mp4 and myvideo.srt).

Viewing Media​

From the Map view, open the Layers panel and toggle the Media layer.

Clicking a marker on the map will expand the media as follows:

Media TypeDescription
Panorama360° interactive viewer
VideoVideo + flight path display
ImageFullscreen image display

Please get in touch if a particular media file does not display correctly.
