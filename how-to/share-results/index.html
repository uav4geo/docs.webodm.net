Share Results

After successfully processing a dataset from the cloud interface, links can be generated to share the results.

tip

Tasks processed in WebODM Lightning but processed outside the cloud interface can be imported by clicking the Import button.

Allowing Annotation Edits​

By default links generated with the Share button do not allow annotations to be persisted or edited.

You can allow edits by clicking the Allow Edits checkbox from the sharing panel.

warning

By enabling Allow Edits anyone with the link will be able to add, remove and edit annotations. We don't recommend sharing such links out in the open internet.

Share Multiple Tasks​

It's possible to share an entire project and all tasks within it, provided that the person one wants to share a project with also has an account on our platform:

  • Have the person access our cloud platform (https://webodm.net/cloud) at least once with their account.

  • From the cloud platform, click the Edit link next to the project that needs to be shared.

  • From the Permissions panel, add the e-mail address of the person.
