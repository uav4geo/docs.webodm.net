Align Datasets

WebODM Lightning has the ability to automatically align two overlapping datasets, without the need for ground control points. This can be useful for comparing changes across time or for compositing larger maps.

From the cloud interface, upload a second task to to be aligned to an existing task in the same project as the first task. Using the Alignment field, select the dataset you want to align the task to. Make sure the tasks are in the same project and that they overlap with each other, otherwise the alignment field will not be displayed.

note

Ground control points are always recommended for highest accuracy results.
